## isnegative

### Instructions

Write a function that prints `'T'` (true) on a single line if the `int` passed as parameter is negative, otherwise it prints `'F'` (false).

### Expected function

```go
func IsNegative(nb int) {

}
```

### Usage

Here is a possible program to test your function :

```go
package main

func main() {
	IsNegative(1)
	IsNegative(0)
	IsNegative(-1)
}
```

And its output :

```console
$ go run .
F
F
T
$
```
